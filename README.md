# Food Review System

A MERN stack web application where:

- Vendors add food items
- Students review foods
- Admin manages users and foods

## Tech Stack

- React Vite
- Node.js
- Express.js
- MongoDB Atlas
- Tailwind CSS

## Features

- JWT Authentication
- Role-Based Access
- Food Reviews
- Image Upload
- Admin Panel
- Change Password

## Run Backend

cd server
npm install
npm run dev

## Run Frontend

cd client
npm install
npm run dev
